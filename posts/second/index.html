Setting up a Website using GitHub Pages

July 6, 2025 · 2 min

This is a guide that will help you in creating your own personal website using Hugo, and deplyoing it with GitHub pages

Prerequisites#

  • Git installed
  • Hugo installed
  • GitHub Account
  • Knowing how to use the terminal

Step 1: Installing Hugo#

I use Arch so i installed it using:

sudo pacman -S hugo

For Debian/Ubuntu users:

sudo apt install hugo

Step 2: Creating you hugo site#

Open terminal and run:

hugo new site<enter name of site> #this allows to install all the main files that hugo allows
cd <name of your site>

Step 3: Choosing theme:#

There are many themes you can choose from on the Hugo site Hugo Themes.I chose PaperMod because it has a very slick and clean look to it.

Downloading PaperMod:

#Make sure you are in the folder you downloaded for creating the hugo site
git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1

This will automatically download the theme PaperMod in your Themes folder

Step 4: Setting up you hugo page:#

Honestly its upto you how you want to configure and all the main configuratins is done through the YAML file (If you chose YAML instead of TOML) such as setting the theme or changing the layout or guiding readers arouond making it friendly

Here are some changes you can make:

Theme = PaperMod #Sets theme for your website in your TOML file (YAML has a different syntax)

profileImage = images/picture.png #adding image

[[params.socialIcons]] #adding useful links to your profiles
    name = "github"
    url = "https://github.com/yourusername"

Step 5: Creating Your First Post#

Make sure you are in your hugo site directory and type:

hugo new posts/first.md #This creates a new post to be written markdown

IMPORTANT To render your site whenever you make a change and view it use :

hugo server #view it at http://localhost:1313

Step 6: Create a GitHub Repository#

use .github.io as the repo name and keep it Public

Step 7: Build the Site#

hugo --minify

This generates a public folder with your sites HTML files

Last Step:#

Push only your pulic folder on to the repo you created

cd public
git init
git remote add origin https://github.com/<your-username>/<your-username>.github.io.git
git checkout -b main
git add .
git commit -m "<add any message>"
git push -u origin main --force

AND NOW YOU ARE DONE YOU HAVE YOUR OWN SITE GENERATED
